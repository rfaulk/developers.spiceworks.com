People Service

People tracked in Spiceworks are typically regular employees of the organization supported by the IT Pro. The people service provides access to information about these users, their devices, any support tickets they may have submitted, and so on.

People

Requests

List people

List all people (IT Pros and end users) in the system:

card.services('people').request('people')
Response
{
  "meta": {
    "total_entries": 166,
    "page_count": 6,
    "per_page": 30,
    "current_page": 1
  },
  "people": [...] // see below for person JSON example
}

Get a single person

card.services('people').request('person', id)
Parameters
Name Type Description
id integer The id of the person
Response
